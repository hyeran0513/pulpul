### pulpul - 개인 간 식물 거래 및 식물 관리 물품 공유 웹서비스 [완성된 프로젝트]

### 사용 기술 스택

- front-end: html, css, javascript
- back-end: Spring(+Thymeleaf), MyBatis
- db: oracle

### 작업 인원

- 김혜란: ui 구성 / 프론트엔드
- 이유리: db관리 / 백엔드 / 프론트엔드
- 윤라경: db관리 / 백엔드 / 프론트엔드
- 구세현: 백엔드
